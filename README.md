# SkyBet-Technical-Test
========
SkyBet Odd Converter

Facts
-----
- version: 1.0.0

Requirements
------------
- Maven :
			
			mvn clean package
			
			or
			
			Run the project: java -jar oddConverter-0.0.1-SNAPSHOT.jar server oddConverter.yml

			Run junit test: mvn test
